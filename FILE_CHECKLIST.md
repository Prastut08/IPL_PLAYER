## 🔧 Customization Quick Links

### Change Colors
**File**: `src/styles/adaptive-questions.css`
**Section**: `:root` CSS variables
```css
--neon-cyan: #00D9FF; /* Primary color */
--neon-blue: #0099FF; /* Secondary */
--neon-gold: #FFD700; /* Accent */
```

### Change Questions
**File**: `src/components/AdaptiveQuestionGenerator.jsx`
**Section**: `ADAPTIVE_QUESTIONS` object
```jsx
const ADAPTIVE_QUESTIONS = {
  initial: ["Your questions here"],
  // ...
}
```

### Adjust Animations
**File**: `src/styles/adaptive-questions.css`
**Section**: `@keyframes` animations
```css
@keyframes glowPulse {
  /* Modify timing here */
}
```
